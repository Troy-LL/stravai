# Design

## Brand

- **Name:** StravAI — pun on Strava; "AI" reads clearly in the wordmark.
- **Positioning:** fitness tracker for AI-assisted dev work; playful but credible.

## Visual direction

- **Activity feed:** card layout modeled on Strava's feed — session title, type icon, key stats (duration, LOC, pace), timestamp, kudos.
- **Profile page:** activity history timeline, weekly totals, personal records.
- **Leaderboard:** sortable table or ranked list; filters for time range and metric exposed in the UI.

## Activity type iconography

| Type | Analogy | Icon metaphor |
|---|---|---|
| Coding | Running | 🏃 |
| Planning | Walking | 🚶 |
| Debugging | Cycling | 🚴 *(future)* |
| Code Review | Swimming | 🏊 *(future)* |

## Key screens (MVP)

1. **Feed** — friends' recent coding sessions with kudos action
2. **Profile** — own activity history and stats
3. **Leaderboard** — weekly LOC total among friends

## MVP UI scope

Match v0.1 in [SPEC.md](SPEC.md): no segments, challenges, or multi-type filters until post-MVP phases.
